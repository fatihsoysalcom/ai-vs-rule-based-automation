# AI vs Rule-Based Automation

Demonstrates the difference between fixed-output traditional automation and adaptive AI automation. Shows how AI can provide varied, context-aware responses to similar inputs, unlike rigid rule-based systems.

## Language

`python`

## How to Run

Save the code as `ai_automation_demo.py` and run it from your terminal using: `python ai_automation_demo.py`
